# LEGADO + QD评论
forked from: https://github.com/gedoor/legado

## v1.0

在Legado的基础上，加入了获取**qd段评**的功能，以按钮形式显示评论数目，并放于每段文字末尾。

只要是QD的小说，都能获取最新的评论。并能够对楼中楼展开，完全对标QD的app。

## v1.1

修复了夜间模式的评论数目和评论弹窗显示不正确的问题，美化了界面和图标

## v1.2

考虑到书源可能多出一些无关段落，导致评论错位，添加了书评按钮双击删除段落的功能

并且删除的段落能够有短期记忆

另外在评论界面加入向前和向后的按钮，可以移动评论的匹配
